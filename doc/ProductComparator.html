Overview

this class lists the differences from 2 products in 4 stages

  1. list all features that are new in the original product

  2. list all features that have been deleted but are still in the copy

  3. do step 1 and 2 for the artifacts of every feature that is included in both products

  4. do step 3 for the passages aswell
